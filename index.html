About the project

Scenes Around Seattle is a project for the xAPI 2017 Fall Cohort. The project uses the A-Frame virtual reality framework to build 360Â° scenes around Seattle. All 360Â° images were taken in Seattle using a 360Â° Ricoh Theta camera. Users are able to choose a specific location in Seattle to visit. Each location has markers with facts about that location.

In the project, xAPI tracks the locations that users visit, as well as the facts that they gather about that location. The project uses a form to help connect to our data with basic authentication. A user can choose to skip filling out the form. A random email address will be generated to authenticate the user, if the user chooses to skip filling out the form. All data is stored in the Yet xAPI Learning Record Store (LRS).
